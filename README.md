## Installation & Setup Instructions

1. Clone down this repository. You will need `node` and `npm` installed globally on your machine.  
2. `npm install`

To Run Test Suite:  

- Via Terminal `npm test`  
- With Cypress GUI `npm run cypress:open`

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`  

## Technologies Used
- ReactJS
- Typescript
- MaterialUI v4
